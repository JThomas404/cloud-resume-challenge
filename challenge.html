AWS Cloud Resume Challenge

The AWS Cloud Resume Challenge is a project designed to help individuals gain hands-on experience with cloud technologies. The challenge involves building a resume and deploying it on the cloud using AWS services. This project demonstrates proficiency in cloud infrastructure, CI/CD pipelines, and infrastructure as code.
